# 🎫 Support Ticket Classification System (FUTURE_ML_02)

## 📌 Project Overview
This project focuses on building a Machine Learning system that automatically classifies customer support tickets and assigns priority levels.

The goal is to simulate a real-world business solution where companies handle large volumes of support requests efficiently.

---

## 🎯 Objectives
- Classify support tickets into categories such as:
  - Billing
  - Technical Issues
  - Account
  - General Queries
- Assign priority levels:
  - High
  - Medium
  - Low
- Improve response time and support efficiency

---

## ⚙️ Tools & Technologies
- Python  
- Pandas  
- Scikit-learn  
- TF-IDF Vectorizer  
- Jupyter Notebook  

---

## 🔍 Methodology

### 1. Data Preprocessing
- Cleaned raw text (lowercasing, removing punctuation)
- Prepared text for analysis

### 2. Feature Engineering
- Converted text into numerical format using TF-IDF

### 3. Model Building
- Used Naive Bayes classifier for text classification

### 4. Model Evaluation
- Evaluated performance using accuracy metric

### 5. Priority Assignment
- Implemented rule-based system:
  - High → urgent keywords  
  - Medium → functional issues  
  - Low → general queries  

---

## 📊 Example Output

Ticket:
"My payment failed and I need help urgently"

Predicted Output:
